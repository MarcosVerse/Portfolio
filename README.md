# Marcos Salles — Portfólio

Portfólio pessoal voltado para tecnologia.

## Stack

| Camada | Tecnologia |
|--------|------------|
| Framework | Vue 3 com Composition API |
| Build tool | Vite |
| Estilização | Tailwind CSS v4 + CSS Scoped |
| Fontes | Syne + DM Mono via Google Fonts |
| Linguagem | JavaScript |

